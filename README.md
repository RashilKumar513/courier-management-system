# 📦 Courier Management System

An admin-only web application to help small courier companies digitize operations with a simple, secure interface.

## 👑 Admin Features
- Admin register & login (secure password handling)
- Manage shipments (create, update, assign)
- Manage customers and drivers
- Track delivery status and history
- View analytics and export data
- Reset admin password

## ⚙️ Tech Stack
- HTML
- Python
- CSS
- JavaScript

## 🚀 Run Locally
```bash
pip install -r requirements.txt
# Run the Python web app (example)
python app.py
```
